## Getting Started

### Clone the Repository

```bash
git clone https://github.com/QASIMAFRID/Flutter.git
```

### Install Dependencies

Navigate to any project folder and run:

```bash
flutter pub get
```

### Run the Application

```bash
flutter run
```

Make sure Flutter SDK is installed and a device or emulator is available.
